# Renegade v1.0.0

Initial public release of Renegade.

Renegade is a Qt/C++ desktop application for adding branded signature headers to source code files.

---

## Included

- Qt 6 Widgets desktop interface
- Dark cyber-style UI
- Recursive project file scanner
- Extension-based file filtering
- Live signature preview
- Smart comment templates by file type
- Signature application engine
- Already-stamped file detection
- Optional `.renegade.bak` backups
- Python shebang and encoding preservation
- Activity log and status feedback
- Windows icon and CMake build support
- Automatic Qt deployment with `windeployqt`

---

## Notes

This is the first public version of the project.

It is recommended to use Renegade on projects tracked with Git or another version control system before applying signatures to many files.
